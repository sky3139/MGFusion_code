F-score是一种综合评价指标 ，它与精确度和recall召回率有关
. 精确率  精确率 = 提取出的正确信息条数 / 提取出的信息条数   召回率 = 提取出的正确信息条数 / 样本中的信息条数 
 F值 = 正确率 *召回率 * 2 / (正确率 + 召回率) （F 值即为正确率和召回率的调和平均值）
